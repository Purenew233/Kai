# Always_remember_me - SillyTavern小说续写器
SillyTavern 专用小说续写扩展，带悬浮球全局面板，支持一键导入本地小说TXT文件、自动按章节拆分、知识图谱构建、AI智能续写。

## 功能特性
- 📖 支持导入本地UTF-8编码的TXT小说文件
- 🔧 可自定义章节拆分正则表达式，适配所有小说的章节标题格式
- 📊 自动生成小说知识图谱，锁定人设和世界观，避免续写OOC
- ✨ 自定义斜杠命令模板，支持角色名、章节内容变量替换
- 🚀 批量自动发送章节，内置防刷屏延迟，稳定不报错
- 🎯 智能续写功能，基于基准章节和知识图谱生成高质量后续内容
- 🎈 悬浮球全局面板，聊天界面随时可用，无需跳转设置页
- 💾 用户配置自动持久化，重启不丢失
- 🔄 支持续写章节链条 JSON 的导入与导出，便于备份、迁移与分享

## 安装方法
### 在线安装（推荐）
1. 打开 SillyTavern，进入「扩展插件」页面
2. 选择「安装扩展」，粘贴本仓库的Git地址：
https://github.com/Neohero521/Always_remember_me
3. 点击「安装」，安装完成后启用扩展
4. 刷新页面，聊天界面左下角会出现悬浮球，同时扩展设置面板也会出现对应功能

### 手动安装
1. 下载本仓库完整代码
2. 解压到 SillyTavern 安装目录下的 `scripts/extensions/third-party/Always_remember_me` 文件夹
3. 重启 SillyTavern 后端服务
4. 进入「扩展插件」页面，找到「小说续写器」，启用扩展
5. 刷新页面即可使用

## 使用方法
1. 点击左下角悬浮球，打开功能面板
2. 点击「选择小说TXT文件」，上传本地的小说文件
3. （可选）修改「章节拆分正则表达式」，适配你的小说章节格式
4. 点击「解析章节」，预览拆分后的章节列表
5. （可选）生成章节知识图谱并合并，保证续写质量
6. 选择续写基准章节，可编辑修改内容，设置续写字数
7. 点击「生成续写章节」，等待AI生成完成
8. 生成完成后，可直接复制内容或发送到聊天对话框
9. 可在「续写章节链条」区域导入/导出续写章节 JSON 文件

## 命令模板说明
- `{{char}}` ：自动替换为当前选中的角色名称
- `{{pipe}}` ：自动替换为章节正文内容
- 默认模板：`/sendas name={{char}} {{pipe}}` ，使用当前角色身份发送章节内容

## 前置要求
- SillyTavern 版本 >= 1.12.0
- 启用内置的 `/sendas` 斜杠命令（默认已启用）
- 配置好可用的AI后端（用于生成知识图谱和续写内容）
